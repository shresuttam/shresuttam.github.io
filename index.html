Pi Box is a set-up box for low-end televisions or standalone monitors that brings “Smart TV” features to it. These features include internet capabilities, hosting and sharing media throughout the local area network, watching streaming media on the television, and more. The 10-foot user interface will be controlled via a remote system that connects to the device via Bluetooth.
